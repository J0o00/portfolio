# Personal Engineering & Research Platform (v1.0.0) 🚀⚡

Welcome to the official repository for **Jovial Joyson's Personal Engineering & Research Platform**. 

This platform is classified as a dual-tier full-stack application:
1. **Engineering Intelligence (`/`)**: A statically generated (SSG), SEO-optimized public portfolio showcase designed for recruiters and engineering managers.
2. **Quantum Control (`/quantum-control`)**: A headless CMS SPA featuring media asset management, AI resume ingestion pipelines, Row-Level Security (RLS), and automated rollback snapshots.
